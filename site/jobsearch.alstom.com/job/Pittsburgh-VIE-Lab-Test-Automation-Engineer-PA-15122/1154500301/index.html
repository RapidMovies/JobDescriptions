Position Overview: The AIT (Automated Integration Testing) team at ALSTOM is seeking a motivated engineer (VIE) to assist in the development, improvement, and testing of our AIT system. The VIE will contribute to the design and development of new features for the AIT system, aimed at achieving full automation of the current Urbalis FLO system integration test cases that are presently executed manually by test engineers.

In this Role, You Will:

  • Participate in the development of Python scripts that automate the conversion of test cases prepared for manual testing into Robot Framework test cases.
  • Collaborate with AIT engineers to design and implement new features that enhance the automation capabilities of the AIT system.
  • Assist in testing and validating the functionality of the AIT system to ensure it effectively performs automated testing of Urbalis FLO systems.
  • Help generate and document comprehensive test reports that summarize testing results and findings from the AIT system.
  • Work on improving the overall functionality and efficiency of the AIT system to facilitate the integration testing process.

Technical Qualifications & Language Requirements:
  • Currently pursuing a degree in Computer Science, Engineering, or a related field.
  • Experience in programming with Python, particularly in automation scripting.
  • Familiarity with software testing processes and methodologies is a plus.
  • Basic understanding of Robot Framework and its functionalities would be advantageous.
  • Bonus if familiar with IBM Rational Quality Manager (RQM) or automation tools.
